5. Web server 설명

같은 컴퓨터에 웹 브라우져와 웹 서버 프로그램을 설치 했다.
127.0.0.1:8080/index.html / 127.0.0.1 의 의미 Internet Protocol Address 의 약자 이다.
8080 의 의미 Port 포트를 의미 한다.
웹 브라우져 자체를 의미하고 웹 서버가 반응하고 요청한 파일을 웹 페이지에 보여준다.
포트 없이 파일을 지정하여 열었을 경우 웹 페이지에 나타나는 것은 똑같으나 주소 표시가 다르다.
웹 브라우져가 직접적으로 파일을 불러 와 웹 페이지에 표시 하기 때문에 다르다.
Hyper Text Transfer Protocol = HTTP 의 의미.
필요한 것은 서로 다른 컴퓨터의 웹 서버와 웹 브라우져 간의 소통이다. 그럼 웹 브라우져는 웹 서비스에게 '인덱스.html' 파일을 달라고 요청해야 한다.
즉, Ip 주소가 필요하다. 이 것을 하려면 웹 서버에 설치된 Ip 주소가 무엇인지 알아야 한다.
컴퓨터 2대가 필요한데, 핸드폰이 있기 때문에 다른 컴퓨터는 필요 없다. 충분하다..
쉽게 사용하기 위해 보통 도메인 주소를 사용하는데, 웹 서버 프로그램을 사용하면 설치된 컴퓨터의 IP가 필요하고
같은 와이파이 상에 있지 않으면 IP 주소를 입력한데도 열리지 않을 수 있다.
